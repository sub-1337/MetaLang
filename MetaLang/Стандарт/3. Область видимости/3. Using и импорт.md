Распаковка области видимости.
```
var libtest = #import("libtest");

#using(libtest); // Добавить в эту область видимости
                 // содержимое libtest
```

Сокращённые записи добавления импорта.
```
~ = #import("libtest");

libtest.functionFromModule();
```

```
#import("libtest");

libtest.functionFromModule();
```
