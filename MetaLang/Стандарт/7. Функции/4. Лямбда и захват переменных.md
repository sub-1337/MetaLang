
В языке есть лямбда функции, которые могут захватывать переменные 
из текущей области видимости.

Данный код напечатает `Hello world`
```
var<str> a = "Hello"
var<fn> test = fn (var str) : copy(var<str> a) { print(a + str); };
test("World");
```
