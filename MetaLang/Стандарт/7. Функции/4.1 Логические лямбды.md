
В языке есть логические лямбды. Они позволяют например 
сортировать удобным образом массивы.
Следующие 2 функции одинаковы:
```
fn func1(var a, var b) -> var<bool>
{
    return a < b;
}

var<fn> func2 = {<};
```

И эти:
```
fn func3(var x) -> var<bool>
{
    return x <= 10;
}
var<fn> func4 = { <= 10};
```
